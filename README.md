# Google Image Searcher for Android

![Video Walkthrough](demo.gif)

## Features
- User can enter a search query that displays a grid of image results
- User can click on "settings" which allows selection of advances search options
- User can configure advance search filters
- Subsequent searches will have any filters applied to the search results
- User can tap on any image in results to see the image full-screen
- User can scroll down infinitely
